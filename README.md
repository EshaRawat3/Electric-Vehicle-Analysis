# Electric-Vehicle-Analysis
The Electrical Vehicle Analysis Dashboard offers an overview of electric vehicle adoption and performance. It visualizes key metrics like total vehicles, average range, top makes, models, and city-wise distribution. This project helps analyze market trends, brand performance, and regional growth, aiding data-driven decisions for sustainability.
